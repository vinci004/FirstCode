Un nouveau paramètre de confidentialité. Les sites Web que vous visitez reçoivent par défaut une demande « Do No Track » quand vous utilisez Internet Explorer. Pour en savoir plus sur ce paramètre, notamment comment le désactiver, voir Utiliser le paramètre Do Not Track dans Internet Explorer.

Remarque au sujet de vos paramètres. Si vous avez effectué une mise à jour d'une ancienne version d'Internet Explorer, vos paramètres pour le filtre SmartScreen, les sites suggérés et les mises à jour des listes de compatibilité ont été enregistrés, sauf si vous avez été invité à les redéfinir. Pour améliorer votre navigation, une quantité limitée de données est collectée, les listes de compatibilité mettent à jour davantage de paramètres de compatibilité et les suggestions de recherche sont disponibles à partir de la barre d'adresses. Pour plus d'informations, notamment sur la modification des paramètres déjà configurés, voir la Déclaration de confidentialité d'Internet Explorer.
